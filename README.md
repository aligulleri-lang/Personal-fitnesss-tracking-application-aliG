# Kişisel Fitness Takip Uygulaması

Bu proje, Python kullanılarak geliştirilmiş terminal tabanlı bir fitness takip uygulamasıdır.
Kullanıcılar antrenmanlarını, beslenmelerini ve vücut ölçümlerini takip edebilir.

## Özellikler
- Kullanıcı kaydı ve girişi
- Antrenman günlüğü (Kardiyo/Ağırlık)
- Kalori takibi
- Veri yedekleme sistemi

## Kurulum
Proje dosyalarını indirin ve terminalde `python main.py` komutunu çalıştırın.
